# HW2-Kalsi ## HW2 - Meeting Summary → Action Items Prototype

This repository contains a small Python prototype that uses the Anthropic 
Claude API to summarize meeting notes and extract action items.

## Files
- app.py — Python script that calls the Claude API
- prompts.md — Three prompt versions with iteration notes
- eval_set.md — Five test cases for evaluating the prototype
- report.md — Business use case, model choice, and deployment recommendation

## How to Run
1. Set your API key: export ANTHROPIC_API_KEY="your-key-here"
2. Run: python3 app.py
